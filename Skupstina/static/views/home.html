Projekat XML web servisi

Web aplikacija radjena u Spring radnom okviru, koja omogucava skupstini da predlaze i prihvata amandmane. Backend je radjen u Spring radnom okviru (java 1.8 - verijza programskog jezika). Za skladistenje podataka koristimo MySQL bazu podataka i Marklogic kao nerealacionu bazu podataka.

Frontend deo realizovali smo pomocu angular RESTFull tehnologije
